
# LARAVEL NEWSPAPER AGGREGATOR

A brief description of what this project does and who it's for


## Installation docker

Run Docker Compose

```bash
  make setup
```
## Access Laravel on Docker

The project running on docker with port :8080

```bash
  127.0.0.1:8080
```
